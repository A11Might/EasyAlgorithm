[479. 加分二叉树](https://www.acwing.com/problem/content/481/)

#### 算法：

*DP* *区间 DP*

**状态表示 - f(i, j)**

- 集合：所有中序遍历是 (i, j) 这一段的二叉树

- 属性：Max

**状态计算 - 集合划分**

以根节点的位置来划分 f(i, j) 表示的集合：

- 根节点在 i

- 根节点在 i + 1

  ...

- 根节点在 k：f(i, k - 1) + f(k + 1, j) + w[k]

记录方案的话就是开一个二维数组 g，存储这个区间的根节点，然后递归还原出答案二叉树。
