**状态表示 - f(i, j)**

- 集合：所有将 [i, j] 合并成一个矩阵（珠子）的方式

- 属性：Max

**状态计算 - 集合划分**

以最后一次合并的分界线来划分 f(i, j) 表示的集合：

- 分界线是 i + 1

- 分界线是 i + 2

  ...

- 分界线是 k：f(i, k) + f(k, j) + w[i] * w[k] * w[j]

综上，f(i, j) = max(f(i, k) + f(k, j) + w[i] * w[k] * w[j])
